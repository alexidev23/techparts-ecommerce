# TechParts Ecommerce

TechParts es un ecommerce construido con React, TypeScript y Vite. El proyecto incluye un sistema de carrito global, filtrado por categorías, navegación dinámica y diseño responsive utilizando Tailwind CSS.

## Características
- React + TypeScript + Vite  
- Gestión global del carrito mediante Context API  
- Filtrado y navegación dinámica de productos  
- Diseño responsive con Tailwind CSS  
- Integración de rutas con React Router  
- Componentes reutilizables para mayor mantenibilidad  

## Scripts
- **npm run dev** – Ejecuta el servidor de desarrollo  
- **npm run build** – Genera la build de producción  
- **npm run preview** – Previsualiza la build  

## Estructura del proyecto
```
src/
 ├─ components/
 ├─ context/
 ├─ data/
 ├─ hooks/
 ├─ pages/
 ├─ router/
 └─ main.tsx
```

## Instalación
```bash
git clone https://github.com/alexidev23/techparts-ecommerce
cd techparts-ecommerce
npm install
npm run dev
```
